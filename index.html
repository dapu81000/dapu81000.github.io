LeetCode458 - Poor Pigs

Description

有很多桶子，每桶都有水，然後其中一桶有毒，豬豬喝了會死
每次測驗每隻豬豬都可以選定喝任意桶水
測驗次數最多n次
問你最至少要幾隻豬豬才能測出哪桶水有毒

Solution1

這題最關鍵的點應該就是要換個角度思考
她問你至少要幾隻豬豬才測得出來
那我們先找固定次數以及固定隻數的豬豬，最多可以測幾桶水
首先假設最多可以測$n$次，然後有$m$隻豬豬，
然後$dp[n][m]$是最多可以測幾桶水
下面歸類一下$dp[n][m]$的各項轉移式

測完一次後，

1.豬豬都沒死
豬豬數一樣，然後少一次機會測
$=> dp[n][m] += dp[n-1][m]$

2.死一隻豬豬
豬豬數少一，然後少一次機會測
注意到這裡要考慮到每隻豬不同，所以要乘上$\binom{m}{1}$
$=> dp[n][m] += \binom{m}{1} \cdot dp[n-1][m-1]$

3.死k隻豬豬
同理，選哪k隻豬豬死
$=> dp[n][m] += \binom{m}{k} \cdot dp[n-1][m-k]$

整理出轉移式之後，大概就可以把整個dp陣列求出來了
複雜度$O(n\cdot m^2)$
假設有k個桶子
就是找到最小的m使得$dp[n][m] \geq k$

Solution2

那如果當n,m越來越大，solution1就可能會超時
這個解法其實是建立在猜測的基礎上XD
觀察轉移式後，發現長得很像展開後的二項式，
所以這邊猜$dp[n][m] = (n+1)^m$
證明的話就用數學歸納法證，不難證就不放過程了(?

最後
假設有k個桶子，我們要找到最小的m
使得$k \leq (n+1)^m$
然後用log再移一下項，可以得到
${log(k) \over log(n+1)} \leq m$
然後ceiling一下，就找到我們要的$m$了
複雜度$O(1)$

LeetCode2334 - Subarray With Elements Greater Than Varying Threshold

Description

給你一串數字和一個threshold
問你能不能從這串數字找出一段連續的數字
使得這段數字都大於等於threshold除以長度

Solution

其實蠻直觀的
對於每個數字
找到前一個和下一個比他小的數字的位置
這樣就可以讓這個數字在這段數字中最小
有點greedy的味道
這裡可以用到monotonic stack來找前一個最小和下一個最小
這邊先上一下找前一個最小的code


大概長這樣
因為我要找前一個最小
所以比現在大的都沒用 因為後面要嘛選自己要嘛選到前面比自己還小的
這就是monotonic的原因
所以這邊可以達成$O(n)$找到前一個最小 下一個最小也是一樣
不過我一開始沒想出來 後來偷看Discuss才會==

然後為了方便計算長度 我找的是前一個最小的下一個數
即$[i-L,i+R]$這段連續數字,$nums[i]$是最小的數,然後讓這串數字盡量長，即讓$L$、$R$最大

LeetCode1473 - Paint House III

Description

大意是有一排房子，然後上漆要錢
定義一坨鄰居為連續的房子，然後漆同樣的顏色
問你有沒有辦法弄出$target$坨鄰居，然後花費要最小

然後一排房子$m$個，總共$n$種顏色的漆
限制大概長這樣
$n{\leq}20,m{\leq}100,target{\leq}m$

大概可以想到$O(nm^2)$的DP
空間也還夠用

Solution

定義一下$dp[i][j][k]$
$dp[i][j][k]$是在前$k$個房子中，第$k$個房子漆上顏色$j$，然後共有$i$坨鄰居的最小花費

dp有兩種轉移
一種是從$i-1$坨，然後跟前一棟不同顏色轉移過來的
另一種是$i$坨，然後跟前一棟同一種顏色
轉移式大概長這樣


最後注意一下題目限制，例如已經漆過的就不能再換顏色

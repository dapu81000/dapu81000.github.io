LeetCode1970 - Last Day Where You Can Still Cross

Description

有一塊大小為$[row,col]$的地
每天都有一個單位的地變成水
問你最晚到第幾天可以從最上面的row走到最下面的row，然後不經過水路

Solution1

利用Disjoint set來合併要變成水的地
合併的時候順便去更新目前整個set的左界右界
然後當左界右界剛好是整塊地的寬度時 就是走不過去了
大致上的複雜度是 $O({row}\cdot{col}\cdot{log(row{\cdot}col)})$

Submit Code

Solution2

另一個可能(?)比較沒那麼麻煩的
二分搜去搜可能的天數 然後判斷該天數可不可以從上面走到下面
所以複雜度一樣是 $O({row}\cdot{col}\cdot{log(row{\cdot}col)})$
大概就是Binary Search + BFS
